## library used

- React
- Bootstrap
- nodejs(for test)

## Run Locally (installation)

### Clone the project

```bash
  git clone https://github.com/alguerocode/postman-clone.git
```

### Go to the project directory

```bash
  cd postman-clone
```

### Install dependencies

```bash
  npm install 
```

### Start the server

```bash
  npm run start
```
